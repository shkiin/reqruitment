== Description ==

The Themes Artist Demo Importer plugin allows you to import demo content for a variety of free themes from [Themes Artist](https://themesartist.com/). Themes Artist is dedicated to creating high-quality, user-friendly WordPress themes. If you want to import demo content from one of the free theme versions of Themes Artist, this is the plugin for you. Please note that this only works with the Themes Artist Themes. This plugin is based on the One Click Demo Import Plugin](https://wordpress.org/plugins/one-click-demo-import). Install the Themes Artist Demo Importer Plugin, One Click Demo Import Plugin and you are good to go.

== Installation ==


1. Goto 'Plugins > Add New',
2. Search 'Themes Artist Demo Importer' and install the plugin,
3. This plugin will also need 'One Click Demo Import' plugin so install that plugin aswell
3. Activate both of the plugin from your Plugins page.
